TransitVis

TransitVis is an interactive 3D iPad app which you can use to examine transit ridership data. It was built for the Urban Data Challenge in the spring of 2013.

Instructions

Start with the video instructions. That provides an overview of the app's operation and purpose.

If've just installed this app, you will need some data. Go read the instructions on downloading data to the app.

If you want to install your own data set, check the blog for details, which should be forthcoming.
